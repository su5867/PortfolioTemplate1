# Portfolio Template 1

This is a customizable portfolio template designed to showcase your projects, skills, and achievements in a clean and professional layout.

## 🚀 Features
- Fully responsive design
- Easy-to-customize HTML and CSS
- Sections for projects, skills, about, and contact
- Smooth animations and transitions
- SEO-friendly structure

  Live Deployment: https://portfolio-template1-six.vercel.app/
